# Nanosync

Ultra-low-latency data replication platform. Single binary, no middleware required.

Stream database changes to your warehouse, message bus, or object store in sub-millisecond latency.

## Installation

```bash
curl -sSL https://install.nanosync.dev | sh
```

Or download a specific version:

```bash
curl -sSL https://install.nanosync.dev | sh -s -- --version v0.2.0
```

## Docker

```bash
docker pull ghcr.io/nanosyncorg/nanosync:latest
```
